When the user says "pull my .claude directory" or refers to syncing/pulling their Claude Code environment, they mean:

- **Repo:** https://github.com/huskerjeff/CaludeCodeSettings.git
- **Local path on other machines:** `C:\Users\huske\.claude` (note: username is `huske` on other machines, `jeffkit` on this one)
- **Clone command for a new machine:** `git clone https://github.com/huskerjeff/CaludeCodeSettings.git ~/.claude`
