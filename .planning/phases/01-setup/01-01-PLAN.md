# Plan: Sync Claude Code Settings from GitHub on a New Machine

## Objective
Replace the existing `~/.claude` folder contents with Jeff's shared settings from
https://github.com/huskerjeff/CaludeCodeSettings.git — without breaking the active
Claude session (credentials must stay in place).

---

## Context
- Repo: https://github.com/huskerjeff/CaludeCodeSettings.git
- The `.credentials.json` file in `~/.claude` must NOT be moved or deleted — doing so
  will end the current Claude Code session.
- The repo contains: `commands/`, `skills/`, `settings.json`, `CLAUDE.md`, `README.md`
- Machine-specific folders (`cache/`, `sessions/`, `plugins/`, etc.) are in `.gitignore`
  and do not need to be preserved.

---

## Tasks

### Task 1 — Back up existing ~/.claude contents

**Action:**
1. Create folder `~/ClaudedotCodeOld`
2. Move everything from `~/.claude` into it **except** `.credentials.json`

```bash
mkdir -p ~/ClaudedotCodeOld
cd ~/.claude
for item in $(ls -A | grep -v '.credentials.json'); do
  mv "$item" ~/ClaudedotCodeOld/ 2>/dev/null
done
```

**Verify:** `ls ~/.claude` should only show `.credentials.json`

---

### Task 2 — Clone repo into a temp folder and copy contents

**Action:**
```bash
git clone https://github.com/huskerjeff/CaludeCodeSettings.git ~/.claude/repo_temp
cp -r ~/.claude/repo_temp/. ~/.claude/
rm -rf ~/.claude/repo_temp
```

**Verify:** `ls ~/.claude` should show: `commands/`, `skills/`, `settings.json`,
`CLAUDE.md`, `README.md`, `.gitignore`, `.credentials.json`

---

### Task 3 — Initialize git and connect to remote

**Action:**
```bash
cd ~/.claude
git init
git config user.name "Jeff Kittrell"
git config user.email "huskerjeff6224@yahoo.com"
git remote add origin https://github.com/huskerjeff/CaludeCodeSettings.git
git pull --rebase origin master
```

**Verify:** `git log --oneline -3` should show recent commits from the repo.

---

### Task 4 — Confirm everything is clean

**Action:**
```bash
cd ~/.claude
git status
```

**Verify:** Output should say `nothing to commit, working tree clean`
